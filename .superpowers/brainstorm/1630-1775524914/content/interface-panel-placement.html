자산 상세 — 인터페이스 패널 위치

현재 connections 탭에 소프트웨어, IP 할당, 담당자, 관련업체, 자산 관계, 별칭이 있습니다.
인터페이스 패널을 어디에 배치할지 선택해 주세요.

A

connections 탭 최상단에 추가

현재 탭 구조 유지. connections 탭 내에서 소프트웨어 위, 첫 번째 섹션으로 인터페이스 배치.
IP 할당 섹션은 그대로 두되, 인터페이스 선택 드롭다운 추가.

connections 탭 — 섹션 순서
🔌 인터페이스 (NEW) — physical, lag, virtual 목록 + CRUD
💿 소프트웨어
🌐 IP 할당 — 인터페이스 선택 후 IP 할당
👤 담당자
🏢 관련업체
🔗 자산 관계
📛 별칭
B

별도 탭으로 분리

기존 4탭(overview, operations, connections, history)에 "인터페이스" 탭 추가.
인터페이스 + IP 할당을 이 탭에 통합. connections 탭에서 IP 할당 제거.

탭 구조
개요 운영 인터페이스 연결 이력
🔌 인터페이스 목록 (physical, lag, virtual)
🌐 IP 할당 — 인터페이스 기반
C

connections 탭 내, 인터페이스 → IP 계층 구조

connections 탭 유지. 인터페이스 섹션을 추가하되, IP 할당 섹션을 인터페이스 하위로 통합.
인터페이스를 선택하면 해당 인터페이스의 IP가 바로 보이는 트리형 구조.
