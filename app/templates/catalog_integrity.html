정합성 관리

제조사 표기 통일, 유사 모델명 정리, 속성 alias 관리를 한 곳에서 처리합니다. 현재 단계에서는 화면 골격과 탭 구조를 먼저 제공합니다.

작업 안내

  • 제조사 정리: 표기가 다른 제조사명을 대표명으로 통일합니다.
  • 제품명 유사 후보: 띄어쓰기, 대시, 축약으로 중복 등록된 모델 후보를 검토합니다.
  • 속성 alias: 도메인, 제품군, 플랫폼 같은 속성값의 alias를 정리합니다.

예정 기능

  • 영향 제품 수 기반 일괄 변경 preview
  • 중복 후보 무시/처리 이력 저장
  • import 단계와 연결되는 alias 정규화

제조사 정리

대표 제조사명과 alias를 관리하고, 영향 제품 수를 기준으로 정리 작업을 진행합니다.

상세 / 일괄 적용

대표 제조사와 alias 목록을 선택하면 여기서 확인할 수 있습니다.

제품명 유사 후보

정규화명과 토큰 유사도를 기준으로 중복 가능성이 높은 제품 후보를 검토합니다.

비교 상세

유사 후보를 선택하면 기준 제품과 후보 제품의 비교 정보가 표시됩니다.

속성 alias

domain, imp_type, product_family, platform 같은 분류/운영 속성값의 alias를 정리합니다.

alias 상세

선택한 alias가 없으면 새 등록 상태로 열립니다.
